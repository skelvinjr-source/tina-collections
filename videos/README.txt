HOMEPAGE BACKGROUND VIDEO
=========================

Add your background video for the homepage hero section.

File Name: homepage-bg.mp4
Location: Place it directly in this /videos folder

Video Requirements:
- Format: MP4 (H.264 codec recommended)
- Resolution: 1920x1080 (Full HD) or higher
- Duration: 10-30 seconds (will loop automatically)
- File Size: 10-20MB recommended for fast loading
- Aspect Ratio: 16:9 (widescreen)

Content Suggestions:
- Fashion models wearing African prints
- Close-up fabric texture shots
- Store/boutique footage
- Lifestyle/lifestyle fashion clips
- Abstract patterns/motion graphics

The video will:
- Autoplay when page loads (muted)
- Loop continuously
- Be covered with a semi-transparent dark overlay
- Scale to cover the entire hero section
- Work on mobile and desktop

Tips:
- Keep the video visually interesting but not too busy
- Ensure important action stays in the center (safe for all screen sizes)
- Use high-quality footage for professional look
- Consider color grading to match your gray theme
- Test on slow connections (keep file size reasonable)

If you don't have a video yet:
1. Use stock footage from sites like Pexels, Pixabay (free)
2. Create a simple slideshow video from product photos
3. Keep the gradient background until you find the perfect video
